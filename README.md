# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes pizza sales data using **SQL Server** and **Microsoft Excel** to uncover valuable business insights, sales trends, and customer ordering patterns.

The analysis focuses on key performance indicators (KPIs), revenue performance, product popularity, and customer purchasing behavior to support data-driven decision-making.

---

## 🛠️ Tools Used

* SQL Server
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

---

## 📊 Key Performance Indicators (KPIs)

* Total Revenue
* Average Order Value
* Total Orders
* Total Pizzas Sold
* Average Pizzas Per Order

---

## 📈 Analysis Performed

### Sales Trends

* Daily Order Trends
* Hourly Order Trends

### Revenue Analysis

* Revenue by Pizza Category
* Revenue by Pizza Size

### Product Performance

* Top 5 Best-Selling Pizzas
* Bottom 5 Selling Pizzas

## 💡 Key Insights

* Large-size pizzas generated the highest revenue contribution.
* Peak ordering hours occurred during lunch and evening periods.
* Sales performance varied significantly across pizza categories.
* A small number of pizza products contributed to a large percentage of total revenue.
* Customer demand showed clear patterns across different days and hours.
