# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes pizza sales data using SQL and Excel to uncover sales trends, customer ordering behavior, and business performance insights.

The analysis focuses on key performance indicators (KPIs), sales trends, best-selling products, and category performance.

---

## 🛠️ Tools Used

- SQL Server
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

---

## 📊 Key Performance Indicators (KPIs)

- Total Revenue
- Average Order Value
- Total Orders
- Total Pizzas Sold
- Average Pizzas Per Order

---

## 📈 Analysis Performed

### Sales Trends
- Daily Order Trends
- Hourly Order Trends

### Revenue Analysis
- Revenue by Pizza Category
- Revenue by Pizza Size

### Product Performance
- Top 5 Best Selling Pizzas
- Bottom 5 Selling Pizzas

## 💡 Key Insights

- Large pizzas generated the highest revenue contribution.
- Peak ordering hours occurred during lunch and evening periods.
- Sales performance varied significantly across pizza categories.
- A small number of products generated a large portion of total sales.
